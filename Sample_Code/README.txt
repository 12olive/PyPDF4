PyPDF4 Sample Code Folder
-------------------------

This will contain demonstrations of the many features
PyPDF4 is capable of. Example code should make it easy
for users to know how to use all aspects of PyPDF4.



Feel free to add any type of PDF file or sample code, 
either by

	1) sending it via email to PyPDF4@phaseit.net
	2) including it in a pull request on GitHub
